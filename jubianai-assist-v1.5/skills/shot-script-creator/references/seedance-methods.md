# Seedance2.0 提示词方法与运镜库

## 一、20 个 Seedance2.0 提示词万能方法

按"基础必学→进阶提升→避坑技巧"分类，覆盖人像、产品、风景、剧情全场景。

### （一）基础必学（8 个）

1. **主体精准定义法**：拒绝模糊表述，必写"核心属性 + 外观特征 + 状态"，权重最高的指令。
   - 示例：20岁东亚淡颜女生，黑长直齐肩发，白色棉麻连衣裙，赤足站在海边，表情温柔。

2. **时序动作写法**：写清动作先后顺序、速度、幅度，避免单点动作和画面卡顿。
   - 示例：女生缓步向前走，手臂自然摆动，发丝随海风轻柔扬起，全程连贯无卡顿。

3. **景运组合法**：每句提示词搭配 1 种景别 + 1 种运镜，不堆砌。运镜写法格式：。新手首选"固定镜头+近景""固定镜头+中景正反打"。

4. **光影具象法**：不用"氛围感拉满"，写清光线类型 + 明暗效果。
   - 示例：黄昏日落逆光，金色丁达尔光效，柔光铺满画面，明暗对比柔和。

5. **画质必加句**：所有场景通用 — 4K 超高清、60fps 高帧率、HDR、细节丰富、无噪点、无模糊。

6. **风格单一法**：单条提示词只加 1-2 个风格词，避免混乱。
   - 示例：复古胶片风格，色彩饱满，颗粒感明显。

7. **图生视频专属法**：必加"与参考图主体完全一致，不修改核心设定"，避免偏离参考图。

8. **负面提示词精简法**：只写核心避坑点。示例：无变脸、无肢体畸形、无穿模、无跳帧。

### （二）进阶提升（8 个）

9. **权重标注法**：用 {} 标注权重（1.0-1.5），核心主体和约束指令权重高于其他。
   - 示例：{30 岁成熟男性 {1.2}，黑色西装}，缓慢整理领带，画面稳定无抖动 {1.1}。

10. **动作细节补充法**：添加动作幅度、速度修饰词，增强画面生动性。
    - 示例：指尖轻轻划过脸颊，动作轻柔缓慢，无僵硬卡顿。

11. **场景氛围具象法**：写清时间 + 环境 + 细节。
    - 示例：深夜城市街头，暖黄路灯，地面湿润反光，霓虹灯光映在墙面。

12. **中英文混写法**：中文写意境，英文写专业镜头/画质术语。
    - 示例：日系清新风格，FPV drone shot（无人机第一视角），4K ultra HD。

13. **多镜头分镜法**：按镜号拆分，每镜标注"景别+运镜+动作"，确保叙事连贯。

14. **强制约束强化法**：必加正向约束，杜绝崩坏。示例：面部清晰无畸变，五官比例自然，同一角色服装全程一致。

15. **节奏控制法**：根据内容添加节奏提示。示例：慢动作定格 3 秒，再匀速恢复正常速度。

16. **素材联动法**：搭配参考图/视频，提示词添加"@图片1 提取造型""@视频1 提取动作节奏"，减少 AI 幻觉。

### （三）避坑技巧（4 个）

17. **运镜不堆砌**：单条提示词最多添加 2 种运镜，避免画面混乱崩坏。
18. **风格不混杂**：不同时添加"复古""赛博朋克""日系"等多种风格，避免画面割裂。
19. **动作不剧烈**：避免"快速跑跳""剧烈舞蹈"，优先选择"缓慢""匀速"动作，提升生成稳定性。
20. **避免模糊表述**：删除"好看、高级、有感觉"等抽象词，全部替换为具体描述。

## 三、Seedance2.0 出片万能公式

```
核心主体（精准描述）+ 时序连续动作（细节+速度）+ 景别 + 运镜组合 + 场景与光影（具体）+ 风格与画质 + 强制约束指令 + 负面提示词
```

**示例（产品展示）：**
透明玻璃瓶装香水，银色金属瓶盖，瓶身浮雕花纹，淡粉色液体 {1.2}，匀速缓慢环绕展示，镜头缓慢推进聚焦瓶身细节，黄昏暖光，复古胶片风格，4K 超高清、60fps 高帧率，无模糊无噪点，画面稳定无抖动 {1.1}，无穿模、无变形。
