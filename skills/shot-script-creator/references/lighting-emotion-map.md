# 光线类型与情绪映射表（AI 短剧专用）

> 整合自专业 AI 视频创作者经验。每类光线对应特定情绪功能，用于画面描述和提示词中精准控制观众感受。

---

## 一、8 种基础光线类型

### 1. 逆光 — Backlight

| 细分类型 | 情绪功能 | 提示词写法 | 适用剧情场景 |
|---------|---------|-----------|------------|
| 英雄逆光 | 塑造英雄感、领袖气场 | 逆光，人物边缘金色轮廓光 | 主角归来、反抗时刻、决战前 |
| 浪漫逆光 | 营造温柔浪漫氛围 | 黄昏逆光，暖调光线包裹人物 | 告白、拥抱、重逢、回忆 |
| 神秘逆光 | 制造身份未知感 | 逆光剪影，人物只剩轮廓 | 反派登场、神秘角色、悬念揭晓前 |

**AI 提示词示例**：
```
逆光勾勒人物边缘金色轮廓，身后光芒四射，面部细节半隐半现
```

### 2. 顶光 — Top Light

| 情绪功能 | 提示词写法 | 适用剧情场景 |
|---------|-----------|------------|
| 压迫感、权力感 | 冷色顶光，眼窝按下，面部阴影加重 | 黑帮老大、幕后掌权者、审讯 |
| 危险感、神秘感 | 顶部单光源，周围环境昏暗 | 地下交易、密室对峙 |

**AI 提示词示例**：
```
冷色顶光从正上方打下，眼窝深陷，鼻翼下方阴影明显
```

### 3. 丁达尔光（体积光）— Volumetric Light

| 情绪功能 | 提示词写法 | 适用剧情场景 |
|---------|-----------|------------|
| 史诗感、神圣感 | 丁达尔光效，空气中可见光柱 | 神殿、教堂、遗迹、森林 |
| 希望感、救赎 | 光线穿透黑暗形成光柱 | 绝境中的希望、信仰时刻 |

**AI 提示词示例**：
```
丁达尔光效，光线穿透黑暗形成可见光柱，空气中微尘漂浮
```

### 4. 底光 — Underlight

| 情绪功能 | 提示词写法 | 适用剧情场景 |
|---------|-----------|------------|
| 诡异感、不安 | 面部从下方被照亮，阴影倒置 | 恐怖场景、鬼故事、噩梦 |
| 疯狂感、失控 | 底光照射，眼周阴影上翻 | 角色黑化、崩溃、发疯 |

**AI 提示词示例**：
```
底光从下颌方向打光，面部阴影倒置，眼窝上方出现不自然亮区
```

### 5. 侧光 — Side Light

| 细分类型 | 情绪功能 | 提示词写法 | 适用剧情场景 |
|---------|---------|-----------|------------|
| 冲突侧光 | 制造紧张感、正邪对立 | 侧光，面部一半亮一半暗 | 对峙、谈判、审问 |
| 内心侧光 | 展现角色内心拉扯 | 侧光勾勒面部轮廓，阴影面占2/3 | 抉择时刻、内心独白 |

**补充说明**：侧光是最常用的情绪光。明暗分割线越锐利，冲突越强；分割线越柔和，内心挣扎意味越浓。

**AI 提示词示例**：
```
单侧光从左侧打来，面部明暗分割清晰，阴影侧眼眶深陷
```

### 6. 硬光（强烈映光）— Hard Light

| 情绪功能 | 提示词写法 | 适用剧情场景 |
|---------|-----------|------------|
| 张力、攻击性 | 强烈硬光，面部阴影锐利 | 战斗准备、对峙升级 |
| 力量感、压迫 | 顶部硬光，肌肉线条明显 | 打斗场景、力量展示 |

**AI 提示词示例**：
```
强烈顶部硬光，面部锐利阴影，肌肉线条被清晰勾勒
```

### 7. 柔光 — Soft Light

| 情绪功能 | 提示词写法 | 适用剧情场景 |
|---------|-----------|------------|
| 温暖治愈 | 柔光铺满画面，阴影几乎不见 | 亲情陪伴、照顾、治愈 |
| 温柔细腻 | 柔光从一侧包裹人物 | 告白、温柔对话、回忆杀 |

**AI 提示词示例**：
```
柔光从窗外洒入，轻轻包裹人物，面部阴影柔和几乎不可见
```

### 8. 蝴蝶光 — Butterfly Light

| 情绪功能 | 提示词写法 | 适用剧情场景 |
|---------|-----------|------------|
| 精致贵气 | 蝴蝶光，鼻下蝴蝶状阴影 | 名媛、贵妇、主角高光登场 |
| 主角感 | 正前方柔光提亮 | 宴会、典礼、舞台 |

**AI 提示词示例**：
```
蝴蝶光从正前方偏上打来，鼻翼下方形成蝴蝶形阴影，五官立体精致
```

## 二、光线选择速查表

按剧情情绪快速选择光线：

| 需要传达的情绪 | 首选光线 | 备选光线 | 说明 |
|--------------|---------|---------|------|
| **英雄/领袖** | 逆光 | 顶光+逆光 | 逆光让人物高大，顶光加重权威感 |
| **浪漫/心动** | 逆光 | 柔光 | 暖调逆光 vs 柔光氛围，逆光更强烈 |
| **神秘/未知** | 逆光剪影 | 底光 | 逆光留轮廓，底光制造诡异 |
| **压迫/权力** | 顶光 | 硬光 | 顶光压制，硬光冲击 |
| **神圣/史诗** | 丁达尔光 | 逆光 | 丁达尔光适合大场景 |
| **恐怖/诡异** | 底光 | 顶光+冷色 | 底光打破日常 |
| **冲突/对峙** | 侧光 | 硬光 | 侧光分界鲜明 |
| **内心挣扎** | 侧光 | 柔光（柔和版） | 侧光分割面部 |
| **温暖/治愈** | 柔光 | 逆光 | 柔光包裹感强 |
| **精致/贵气** | 蝴蝶光 | 侧光+柔光 | 蝴蝶光最正式 |
| **狠厉/攻击** | 硬光 | 顶光 | 硬光有冲击力 |

## 三、女频 AI 短剧专属光影配方

来源：山灵制造局——女频爆款短剧画面分析

### 光影锁定三件套

| 要素 | 作用 | 提示词 |
|------|------|-------|
| 轮廓光 | 让人物从背景分离 | rim light, edge lighting |
| 发丝光 | 头发边缘透光显精致 | hair light, translucent hair edges |
| 小景深 | 背景虚化突出主体 | shallow depth of field, bokeh background |

### 前景氛围法

用环境元素做前景遮挡，从"精致"升级为"有氛围感的精致"：

| 场景 | 前景遮挡物 |
|------|-----------|
| 室内对话 | 窗帘/纱帘/门框 |
| 花园/户外 | 花枝/树叶/草叶 |
| 古风内景 | 屏风/珠帘/窗棂 |
| 现代都市 | 百叶窗/玻璃反光/灯罩边缘 |

### 女频爆款画面公式

```
轮廓光 + 发丝光 + 小景深 + 环境前景 = 有氛围感的精致画面
```

## 四、光线与景别的配合

| 景别 | 光线表现重点 | 适合的光线类型 |
|------|------------|-------------|
| 远景 | 整体光线氛围、环境光 | 丁达尔光、逆光大场景 |
| 全景 | 人物的光影轮廓与环境关系 | 逆光、硬光 |
| 中景 | 人物的姿态与光影分布 | 侧光、顶光 |
| 近景 | 面部情绪与光线细节 | 侧光、蝴蝶光、柔光 |
| 特写 | 极致的明暗对比/局部光效 | 侧光、硬光、底光 |
| 极致特写 | 瞳孔倒影、高光细节 | 任何光线类型均可 |

## 五、光线类型在画面描述中的应用

在画面描述的**视觉重点**和**氛围暗示**中使用，而非在风格字段单独列出：

**正确用法**（融入画面描述）：
- "侧光从左侧打来，陈峰脸上明暗分明，一半脸藏在外套阴影里"
- "逆光勾勒小七的轮廓，夕阳在她发丝边缘镀上金色"
- "丁达尔光柱从教堂穹顶投射而下，尘埃在光中浮动"

**错误用法**（抽象空洞，不具体到光型）：
- ❌ "光影氛围感强" 
- ❌ "电影级打光"
- ❌ "光线很美"

**光线写法格式（主光+辅光）**：
```
主光→场景图[物体]的[光源名]从画面[方位]照入，[视觉效果]
辅光→场景图[物体]的[光源名]从画面[方位]补入，[视觉效果]
```
主光定基调，辅光补层次。单光源场景辅光写「辅光：无」。每束光必须锚定场景图具体物体，禁止无源光。禁止抽象词（"氛围感""电影光"），必须写具体视觉效果。

**光线类型字段应出现在**：
1. 运镜字段确认机位后，光线字段按主光+辅光结构书写
2. 视觉重点字段中的核心视觉元素
3. 但不单独作为风格字段
